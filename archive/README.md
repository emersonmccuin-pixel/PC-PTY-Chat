# Archive

Non-runtime project material moved out of the app surface.

- `docs/` - historical plans, audits, handoffs, runbooks, and investigations.
- `tests/` - former unit and Playwright suites.
- `github/` - former GitHub Actions workflows.
- `scripts/` - ad-hoc migration and investigation utilities.
- `project-companion/` - old root companion prompt material.
- `temp/` - ignored local scratch files, screenshots, logs, mockups, local tool state, and old fixtures.

The root repo now keeps the app runtime, build inputs, templates loaded by the app, live local state, and minimal operating metadata.
